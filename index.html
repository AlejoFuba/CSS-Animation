Bootstrap

Framework CSS con utilidades, componentes y un sistema de grid rápido para construir interfaces responsivas.

¿De qué trata?

Bootstrap ofrece un conjunto de componentes preconstruidos (botones, tarjetas, navbar), utilidades (margen, padding, display) y un grid responsivo de 12 columnas. Acelera el desarrollo UI especialmente en prototipos y admin panels.

Cómo usarlo

  1. Incluir CSS y JS de Bootstrap (CDN o instalando via npm).
  2. Usar clases semánticas: container, row, col, y utilidades como d-flex, justify-content-between.
  3. Personalizar con SASS o sobrescribiendo variables si se requiere identidad de marca.

Ventajas

  • Rápida productividad y prototipado.
  • Consistencia entre componentes y amplia documentación y comunidad.

Desventajas

  • El resultado puede verse genérico si no se personaliza.
  • Incluye CSS y JS que podrías no necesitar.

Nota: Para usar componentes JS (modal, dropdown) necesitas el bundle JS de Bootstrap o Popper si tu versión lo requiere.

Bootstrap es ideal para acelerar el desarrollo, pero siempre verifica peso y personalización para proyectos en producción.
