¿De qué trata?

En CSS hay dos herramientas principales para animación: transition (interpolación entre estados) y @keyframes con animation (animaciones más complejas y repetitivas).

Cómo usarlo

  • transition: define propiedades a interpolar, duración y timing-function. Ej: transition: transform 200ms ease;
  • @keyframes: define pasos de la animación, luego se aplica con animation (duración, iteración, dirección).

Ventajas

  • Mejor experiencia de usuario cuando se usan para estados y feedback visual.
  • Buen rendimiento si se animan propiedades transform/opacity (GPU-friendly).

Desventajas

  • Animaciones excesivas o largas pueden distraer y afectar accesibilidad.
  • Algunas propiedades provocan reflow (ej: width, height) y deben evitarse para animaciones fluidas.

/* transition */
.box{transition:transform .28s ease, box-shadow .28s ease}
.box:hover{transform:translateY(-8px);box-shadow:0 12px 28px rgba(0,0,0,0.08)}


/* keyframes */
@keyframes pulse{0%{transform:scale(1)}50%{transform:scale(1.2)}100%{transform:scale(1)}}
.element{animation:pulse 1.6s infinite}
